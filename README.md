# FM Transmitter
This repository contains FM Transmitter schematics, PCB layout, output files ( Designed using Altium ) [2 Layer PCB]

Used ICs:
* Voltage Controlled Oscillator: MAX2606
* Microphone Pre-amplifier: MAX4467
